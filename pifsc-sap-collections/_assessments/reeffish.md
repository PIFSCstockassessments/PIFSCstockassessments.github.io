## 2019 Guam

The 2019 assessment comprised of 12 species of reef fish using the growth-type-group length-based
spawning potential ratio (GTG LBSPR) approach.

The [StepwiseLH R package](/packages/stepwiseLH/) was used to generate some of the life history parameters used in this effort and the [TMB.LBSPR R package](/packages/tmb-lbspr/) was used to run the model itself.

* Links to the 2019 Guam Reeffish Assessment Report and the GitHub Source Code is pending availability.

## 2017 Hawaii

The 2017 assessment comprised of 27 species of reef fish using a mean-length mortality model and a
population model to estimate spawning potential ratios (i.e. stock status).

The StepwiseLH approach was also used to generate life history parameters for some species. All of these steps were implemented in a
Java-language program available upon request.

* Links to the 2017 Hawaii Reeffish Assessment Report and GitHub Source Code Repo is pending availability.
